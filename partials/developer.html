View Code

Console application that allows a user to interact with a basic product ordering database. Users can login/logout, view product popularity, add items to an order and finalize the order by checking out. This app was built in C# using the .NET framework.

View GigGetter

Web Application to promote the services of a musician/singer in a music business market as well as search for potential gigs posted by producers/engineers. This app was built in Angular JS using Grunt and Boostrap.

Play Game

Web Application demonstrating prototypal inheritance that allows the user to select a candidate and "debate" them until a winner is declared. This app was built with Javascript using Grunt and custom HTML5/CSS3.

Track Family Members

Web App that allows the user to build a database of their familiy members. This was built with Javascript using custom HTML5/CSS3.

Check Out Order Form

Web Form that allows the user to build a custom sandwhich. This was built with Javascript using custom HTML5/CSS3.
